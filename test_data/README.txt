This folder contains test data files (.smd,.txt) for use in tests and examples.

Rough does not currently read .smd files, these are used in reference software 
packages and kept here for future reference. 

All .smd files and associated .txt files were obtained from: 

http://resource.npl.co.uk/softgauges/Simu_Feature.htm
http://resource.npl.co.uk/softgauges/Simu_Proc.htm
http://resource.npl.co.uk/softgauges/Masterpiece_Measure.htm

All reference test results for ISO profile parameter values were calculated from the 
softgauges program provide by the npl:

http://resource.npl.co.uk/softgauges/Ref_Algo_Overview.htm
